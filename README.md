# 🏭 Manufacturing Yield & Downtime Dashboard
**comprog final project**

## 📌 Project Overview
This dashboard is designed for plant floor managers to monitor production processing times and predict potential downtime. Using Linear Regression and Statistical Process Control (SPC) logic, the tool identifies anomalies in real-time.

## 🛠️ Tech Stack
* **Language:** Python
* **Dashboard:** Streamlit
* **Visualization:** Plotly (Interactive Control Charts)
* **ML Model:** Linear Regression (Scikit-Learn)

## 📊 How to Run
1. Install requirements: `pip install -r requirements.txt`
2. Launch dashboard: `streamlit run project.py`

## 📈 Executive Impact
By flagging "out-of-control" points using 3-sigma limits, this tool allows for predictive maintenance, helping to maximize yield and reduce unscheduled machine downtime.
